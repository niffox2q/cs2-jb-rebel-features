### Модуль Rebel Features для Jailbreak Core  

Изменяет характеристики и визуальные эффекты при активации статуса бунта.

При становлении бунтующим игрок получает:
- увеличенное здоровье и броню
- изменённую скорость передвижения
- кастомный цвет модели
- проигрывание глобального и локального звука

Все параметры берутся из конфигурационного файла `addons/configs/Jailbreak/rebel_features.ini` и применяются автоматически при срабатывании статуса бунта.

### Установка:
Распаковать в `game/csgo/addons/`

### Требования:
- [Jailbreak Core](https://discord.gg/WkTwuKe8zy)
- [Utils](https://github.com/Pisex/cs2-menus)
